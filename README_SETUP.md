# تطبيق محمد عبد العظيم الطويل الإسلامي 🕌

## خطوات الرفع على GitHub وبناء APK

### 1️⃣ أضف صورة الوالد
- سمّي الصورة: `father_photo.jpg`
- احطها في مجلد: `assets/`
- بتحل محل الملف الموجود

### 2️⃣ ارفع على GitHub
```
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git push -u origin main
```

### 3️⃣ GitHub Actions هيبني APK تلقائياً
- روح لـ Actions في الـ repo
- انتظر ~10 دقايق
- حمّل الـ APK من Artifacts
